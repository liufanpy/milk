# 全局重构 — 实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Header-Detail 分离 + 全局统一 ID + 精简流水表 + 导入导出全覆盖

**Architecture:** 新建 `documents` 基表统一生成全局 ID，订单表 PK 改为 `document_id`；每类订单独立 items 表存品项；StockMovement / Transaction 精简至最少字段。所有导入统一两步模式（preview + confirm），导出数据源从 items 表出。

**Tech Stack:** Python/FastAPI + SQLAlchemy + SQLite, React/TypeScript + Zustand

```python
def create_document(db, doc_type: DocumentType) -> Document:
    prefix = PREFIX_MAP[doc_type]
    doc = Document(doc_type=doc_type)
    db.add(doc)
    db.flush()
    doc.order_number = next_order_number(db, Document, prefix, doc_type)
    return doc
```

```python
# backend/app/services/order_number.py
from datetime import date
from sqlalchemy.orm import Session

def next_order_number(db: Session, model, prefix: str, doc_type=None) -> str:
    today = date.today().strftime("%Y%m%d")
    full_prefix = f"{prefix}{today}"
    q = db.query(model).filter(model.order_number.like(f"{full_prefix}%"))
    if doc_type is not None:
        q = q.filter(model.doc_type == doc_type)
    last = q.order_by(model.id.desc()).first()
    if last and last.order_number and len(last.order_number) == 14:
        seq = int(last.order_number[-4:]) + 1
    else:
        seq = 1
    return f"{full_prefix}{seq:04d}"
```
